#  Weather Dashboard App

##  Project Overview

This is a web application that allows users to search for real-time weather information of any city.

---

## 🎯 Purpose

To practice API integration and JavaScript.

---

## API Used

* OpenWeather API
* https://openweathermap.org/api

---

##  Features (Planned)

* Search weather by city
* Show temperature
* Show weather condition
* Show humidity
* Show wind speed

---

## 🛠️ Technologies

* HTML
* CSS
* JavaScript

---

## ⚙️ Setup

1. Get API key from OpenWeather
2. Add it in script.js
3. Open index.html
